# KENJI GUTIERREZ JIMENEZ - Portfolio

AI・DX推進エンジニア / プロダクトマネージャーのポートフォリオサイト

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Build**: Vite
- **3D Graphics**: React Three Fiber + drei
- **Animation**: Framer Motion
- **i18n**: react-i18next (日本語 / English / Español)
- **Styling**: CSS Variables + Glass Morphism

## Features

- Apple風ミニマルデザイン
- 3Dフローティングジオメトリ & パーティクル
- インタラクティブなスキル可視化（3Dオーブ）
- プロジェクトケーススタディモーダル
- 多言語対応（3言語）
- レスポンシブ対応
- パフォーマンス最適化（コード分割）

## Getting Started

```bash
cd kenji-portfolio
npm install
npm run dev
```

## Build

```bash
npm run build
```
